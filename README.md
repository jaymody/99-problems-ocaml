Solutions for [ocaml.org/problems](https://ocaml.org/problems).

### Usage
Dependencies:
```shell
opam switch create . -w
```

Build project:
```shell
dune build
```

Resources:
- [Real World OCaml](https://dev.realworldocaml.org/index.html)
